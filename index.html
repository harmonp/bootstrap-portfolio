Customizable Productivity Timer

This Javascript Productivity Timer is both interactive and customizable. As time passes, the circle slowy fills up like the bottom of a sand measure. The timer defaults to 52-17, based on the University of Toronto study that found productive employees worked best when they took 17-minute breaks for every 52-minutes of work. Each time can be customized in minute increments.

  • Site: Timer
  • Date: September 2015
  • Skills: HTML, CSS, Javascript
